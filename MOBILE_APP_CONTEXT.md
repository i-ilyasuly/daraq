# Daraq Мобильді/Чат Қосымшасының Контексті (Context & Integration State)

Бұл құжат **Daraq** жобасы мен оның үшінші жақты интерфейстері (ChatGPT стиліндегі жаңа ChatGPT-UI немесе Мобильді қосымшасы) арасындағы байланысты, қазіргі даму кезеңін және маңызды техникалық ережелерді сақтауға арналған.

---

## 1. Контекст және Архитектуралық Ағым

* **Дизайн мен Визуалды Көрініс:** Басқа AI Studio Build жобасында ChatGPT және заманауи чат қолданбалары сипаттас толықтай визуалды дизайн дайындалды. Басты назар — мобильді қолданысқа ыңғайлы ChatGPT-like интерфейс.
* **Бэкенд пен Интеграция:** Бұл бэкенд (Daraq RAG сервері) GitHub-қа пуш етілген. Әрбір өзгеріс Cloud Run-ға автоматты түрде деплой етіліп отырады.
* **API ретінде қолдану:** Сол Cloud Run ішіндегі `run.app` мекенжайы жаңадан жасалған визуалды немесе мобильді интерфейске негізгі API сервері (Endpoint) ретінде қызмет етеді.

---

## 2. Ағымдағы Даму Кезеңі (Current Integration Stage)

1. **Phase 1: Дизайн дайындау** — Орындалды (Жаңа ChatGPT-like интерфейс дизайны мен мобильді құрылымы басқа жобада дайын).
2. **Phase 2: Бэкендпен байланыстыру (API Integration)** — Басталды. Дизайнды біздің Cloud Run-дағы `run.app` API мекенжайымен кіріктіру жоспарлануда.

---

## 3. Маңызды Шектеулер мен Ережелер (Crucial Constraints)

* **❌ ДАУЫСТЫ ОЙНАТУДЫ ҚОСПАУ:** Біз жаңа интерфейске дауысты ойнату (Audio TTS / Voice player) мүмкіндігін **әзірге мүлде қоспаймыз**. Оған қатысты ешқандай батырмалар, ойнатқыштар немесе дауыстық визуализация ағымы әзірге жасалынбауы керек. Тек мәтіндік чатқа назар аударылады.
* **Серверлік API Бағыттары:** Тұтынушының жаңа чат интерфейсіне сұрақ жіберіп, RAG жауап алуы үшін мынадай endpoint-тер қолданыла алады:
  - `POST /api/chat` немесе Телеграм вебхук үшін қажетті API ағымдары.
* **Таза Дизайн:** Фронтенд жобасында артық техникалық деректер, сервер кодтары немесе статус белгілері («ONLINE», «Active») көрсетілмей, өте таза ChatGPT стилінде сақталады.
